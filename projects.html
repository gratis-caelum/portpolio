Based RAG System - Fairy Tail AI Generator

Project Overview

Fairy Tail AI Generator is a project that generates fairy tales based on user requests using RAG (Retrieval-Augmented Generation) System. It analyzes user requests and generates customized fairy tales.

Main Features

  • RAG (Retrieval-Augmented Generation) System
  • ChromaDB Vector Database
  • LangChain Framework
  • Fairy Tail Generator

TABA Program - AI-based Visual Impaired Person Assistance System

Project Overview

TABA Program is a project that develops an AI-based real-time safety and communication assistance system for visually impaired people. I participated as an AI team member.

Main Features

  • Situation Recognition and Description of Surrounding Environment
  • Voice Recognition-based User Interface
  • Location-based Service Integration

Used Technologies

  • Python, TensorFlow, PyTorch
  • Computer Vision (OpenCV, YOLO)
  • Voice Recognition and Synthesis (Flutter Library STT/TTS)
  • Mobile Application Development
