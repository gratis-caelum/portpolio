캡스톤디자인 - RAG System 기반 동화 생성

프로젝트 개요

LangChain + ChromaDB를 활용한 RAG System을 적용한 동화 생성 프로젝트입니다. 사용자의 요구사항을 분석하여 맞춤형 동화를 생성합니다.

주요 기능

  • RAG (Retrieval-Augmented Generation) 시스템을 활용한 정보 검색
  • ChromaDB를 통한 벡터 데이터베이스 구축
  • LangChain 프레임워크를 이용한 LLM 연동
  • 사용자 요구에 맞는 동화 생성

사용 기술

  • LangChain, ChromaDB, Python
  • Flask/Django (백엔드)
  • React (프론트엔드)

DKU경진대회 - 스마트 팩토리 전력 사용량 예측

프로젝트 개요

스마트 팩토리 Time-series data를 활용하여 전력 사용량 예측 모델을 개발하는 프로젝트입니다. 데이터 전처리, 탐색, 분석 역할을 담당했습니다.

주요 기능

  • 시계열 데이터 분석 및 전처리
  • 다양한 머신러닝 모델 학습 및 비교
  • 예측 모델 평가 및 최적화
  • 결과 시각화 및 보고서 작성

사용 기술

  • Python, pandas, NumPy, scikit-learn
  • 시계열 모델 (ARIMA, Prophet)
  • 딥러닝 모델 (LSTM, GRU)
  • 데이터 시각화 (Matplotlib, Seaborn)

LevelDB 연구

프로젝트 개요

Key-value DB 종류 중 많이 사용되는 LevelDB/RocksDB를 공부 후 현재 LevelDB와 관련된 가설-실험-결론 과정을 통해 연구를 진행중입니다. hot-potato라는 새로운 write cache 개념을 가진 in-place update 변환 연구를 수행하고 있습니다.

연구 주제

  • LevelDB의 성능 최적화 방안 연구
  • hot-potato write cache 개념 도입
  • in-place update 메커니즘 구현
  • 성능 개선 효과 측정 및 분석

사용 기술

  • C++, LevelDB, RocksDB
  • 데이터베이스 벤치마킹 도구
  • Linux 시스템 프로그래밍
